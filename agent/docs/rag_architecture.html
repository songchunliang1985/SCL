检索策略对比

🔍 rag_search()

单角度精确检索

适合：精确查找特定条款、数字、术语

  • 1
    原始 query → 向量召回 Top-20
  • 2
    同一 query → BM25 召回 Top-20
  • 3
    RRF 融合两路结果
  • 4
    CrossEncoder 精排
  • 5
    返回 Top-4

🤖 rag_ask() 推荐

Agentic 多角度检索

适合：复杂问题、跨文档推理

  • 1
    _expand_queries() 扩展出 ≤5 个查询
  • 2
    每个 query 各自向量+BM25 检索
  • 3
    RRF Meta-Fusion 跨列表去重融合
  • 4
    CrossEncoder 精排
  • 5
    返回 Top-6 + queries_used
查询扩展规则（无需 LLM，纯规则）
规则 1
去除疑问词：请问、麻烦 → 陈述形式
请问主角是谁？→ 主角是谁
规则 2
连词拆分：和、以及、还有 → 子问题
A和B → ["A", "B"]
规则 3
jieba 提取 Top-5 关键词作为短查询
jieba.analyse.extract_tags()
→ ["关键词1", "关键词2", ...]
05

核心算法

🔀
RRF 融合算法
Reciprocal Rank Fusion
score(d) = Σi 1 / (k + ranki(d))
k=60（经验平滑参数），rank 从 1 开始
# 多路结果融合，天然去重
def _rrf_fusion(ranked_lists, k=60):
    scores = {}
    for ranked_list in ranked_lists:
        for rank, (doc, meta) in enumerate(ranked_list, 1):
            doc_id = md5(doc)  # 内容去重
            scores[doc_id] += 1 / (k + rank)
    return sorted(scores, key=score, reverse=True)
优势：不依赖分数量纲，向量分数和 BM25 分数可直接融合
🎯
CrossEncoder 重排
bge-reranker-base
relevance = sigmoid(CrossEncoder(query, passage))
Sigmoid 归一化: 1 / (1 + e-score)
# 比向量相似度更精准的语义匹配
pairs = [(query, chunk) for chunk in candidates]
raw_scores = reranker.predict(pairs)
scores = 1 / (1 + np.exp(-raw_scores))

# 过滤低分
results = [(s, doc) for s, doc
           in zip(scores, candidates)
           if s >= 0.1]
与向量检索区别：CrossEncoder 同时看 query+passage，精度更高但速度慢
🀄
BM25 中文检索
BM25Okapi + jieba
# 分词 + 过滤停用词
def _tokenize(text):
    tokens = jieba.lcut(text)
    return [t for t in tokens
            if t not in STOPWORDS
            and len(t) > 1]

# 中文停用词举例
STOPWORDS = {"的", "了", "是", "在",
             "有", "和", "与", "等", ...}

# 每次 ingest/delete 后重建
_bm25 = BM25Okapi([_tokenize(d) for d in corpus])
🔄
Lazy-Load 单例模式
延迟初始化，避免启动慢
# 全局单例，首次调用时初始化
_client   = None  # ChromaDB client
_col      = None  # Collection
_reranker = None  # CrossEncoder
_bm25     = None  # BM25Okapi

def _get_collection():
    global _client, _col
    if _col is None:
        _client = PersistentClient(DB_PATH)
        _col = _client.get_or_create_collection(
            "rag_docs_bge",
            embedding_function=SentenceTransformerEmbeddingFunction(
                model_name="BAAI/bge-small-zh-v1.5")
        )
    return _col
