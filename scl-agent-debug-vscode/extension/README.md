# SCL-Agent-debug-VScode (extension)

让 Claude Code 通过 MCP 直接驱动 VS Code 调试器：打断点、单步、看变量、
求值表达式 —— 全部由 Claude 自动调用。

完整文档（含安装手顺、原理图、故障排查、支持的语言）见仓库根
[README](https://github.com/songchunliang1985/SCL)。

## 命令

- `SCL-Agent-debug: Setup Claude Code Integration` —— 一次性写入 MCP 配置
- `SCL-Agent-debug: Remove Claude Code Integration` —— 移除配置
- `SCL-Agent-debug: Toggle (pause/resume)` —— 暂停 / 恢复
- `SCL-Agent-debug: Show Log` —— 打开扩展日志
